# Weihnachtswusch

Statische, barrierefreie Webseite für das Weihnachtsmusik-Projekt **Weihnachtswusch**.

## Über

Die Geschichte vom kleinen Weihnachtswunsch, der sein "n" verlor und als Weihnachtswusch Weihnachtslieder durch die Nacht trällert.

## Technik

- Statisches HTML/CSS (kein Framework, kein JavaScript-Build)
- Gehostet auf Vercel
- WCAG 2.2 AA konform
- Keine Cookies, kein Tracking

## Barrierefreiheit

- Semantisches HTML5
- Skip-Link
- aria-label für Navigation
- aria-current für aktive Seite
- Kontrastgeprüfte Farben (WCAG AAA für Text, AA+ für Links)
- Responsive Design
- Reduced Motion Support
- High Contrast Mode Support
- Alt-Texte für alle Bilder

## Entwicklung

Einfach die HTML-Dateien im Browser öffnen oder einen lokalen Server starten:

```bash
npx serve .
```
